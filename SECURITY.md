# Security Policy

You found an issue? Report it

## Please DON'T
- Post it publicly in an issue 
- Test it on real users or real data
- Hold it for ransom

## Please DO
Email me quietly: **ckazros@kozzyx.org**

Tell me:
- What's broken
- How to reproduce it
- How scary you think it is

## What happens next
- I'll try to reply within a few days
- I'll confirm it, fix it, and credit you if you'd like - or keep you completely anonymous. Your call.

## Supported versions
Whatever's live right now. There is no enterprise LTS support tier.

Thanks for being one of the good ones.
